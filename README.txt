SDL SNAKE
================================================================================

It's Snake, coded with SDL (and SDL_mixer) in C!

By 44D Again Again (44d-again-again.itch.io)

CONTROLS
--------------------------------------------------------------------------------

  * Arrow keys to move
  * Enter key to select


GAME NOTES
--------------------------------------------------------------------------------

EASY is the slowest snake; each food gives 1 point

NORM is a faster snake; each food gives 2 points

HARD is the fastest snake; each food gives 4 points
